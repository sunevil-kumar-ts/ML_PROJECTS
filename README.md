### CLASSIFICATION PROJECT

# **Breast Cancer Analysis and Prediction Using Data Analytics**

## **Brief Overview & Conclusion**

This project focuses on analyzing breast cancer data to classify tumors as **benign or malignant** using machine learning techniques.
Early detection is critical, and data analytics helps improve diagnostic accuracy and decision-making.

Multiple models were evaluated using accuracy, confusion matrices, and classification reports. **KNN, SVM, 
and Gaussian Naive Bayes achieved the best performance (96% accuracy)** with strong and balanced predictions. **Logistic Regression and Decision Tree** showed 
slightly lower performance (**94% accuracy**), while **Random Forest performed poorly (46%)**, indicating it was not suitable without further tuning.

**Final Insight:**
**KNN, SVM, and SVM-based approaches are the most reliable for this dataset, providing accurate and consistent predictions for breast cancer classification.**
